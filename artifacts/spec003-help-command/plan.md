### Watch Out
- There is no host CLI; `help` must be a conversational trigger handled by ARTHUR (FR-102 prohibits host-level registration).
- Team roster, README, and agent frontmatter are three possible sources of truth; prefer frontmatter to avoid drift.
- Include `.github/agents/temps/` explicitly when globbing or active temp agents will be missed.
- Parsing frontmatter increases token cost; keep output terse (one-line summaries per agent) to control prompt size.
- Make changes to `.github/agents/arthur.agent.md` small and easily revertible; ARTHUR is the dispatcher and mistakes can break routing.
